# Texts Processing and Analytics Data on Medium Articles

### **Feature :**
1. Phyton Language
2. Text Processing
3. Data Analytics
4. Graph Preview

### Program Goals :
- [x] Knowing the Strategy for Writing Article Titles on medium platforms so that articles will appear more often by user searches

### Analytics Process:
- [x] Read CSV
- [x] Data Cleaning
- [x] Collect All Title
- [x] Text Processing
    + Case Folding
    + Tokenizing 
    + Filtering (Stopword Removal)
    + Steming Process
- [x] Visualisation
    + Line Chart
    + BoxPlot
    + Chart Bar
